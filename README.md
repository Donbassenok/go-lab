## Встановлення та запуск
### 1. Склонуйте репозиторій
```bash
git clone <url-вашого-репозиторію>
cd go-lab
```
### 2. Встановіть залежності
```bash
go mod tidy
```
### 3. Налаштуйте оточення
Створіть файл `.env` у корені проєкту та додайте рядок підключення до бази:
```Plaintext
DB_URL=data.db
```
### 4. Запустіть сервер
```bash
go run ./cmd/server/main.go
```
Після запуску сервер буде доступний за адресою `http://localhost:8080`. При першому запуску автоматично створиться файл бази даних та застосуються міграції.
## Тестування
Для запуску всіх юніт-тестів виконайте команду:
```bash
go test ./... -v
```
## API Ендпоінти
|Метод|Ендпоінт|Опис|
|:-----------|:------------|:------------|
|POST|`/plants`|Створити нову рослину|
|GET|`/plants`|Отримати список усіх рослин|
|GET|`/plants/{id}`|Отримати рослину за ID|
|PUT|`/plants/{id}`|Повне оновлення даних|
|PATCH|`/plants/{id}`|Часткове оновлення (напр. віку)|
|DELETE|`/plants/{id}`|Видалити рослину|
## Тестування через Postman
У корені проєкту знаходиться файл `postman_collection.json`. Імпортуйте його в Postman, щоб швидко протестувати всі доступні методи.
